# Personacle

記憶と成長する人格を持つ、ローカルAIキャラクター育成アプリ。

会話の内容はキャラクター(ペルソナ)の「記憶」として積み重なり、性格や関係性が経験によって少しずつ変化します。ペルソナ同士を自律的に会話させて、その様子を観察することもできます。**すべての処理と保存はあなたのPCの中で完結し、外部にデータを送信しません。**

## 主な機能

- **ペルソナ作成**: 性格・口調・価値観を自由に設定。5つの性格軸(社交性・共感性・慎重さ・自己主張・明朗さ)を持つ
- **1対1チャット**: 記憶と人格を反映したストリーミング応答
- **記憶システム**: 会話を終えると重要な内容が記憶として抽出され、次回以降の会話で想起される(意味検索＋新しさ＋重要度のハイブリッド想起)
- **人格の成長**: 会話の経験で性格と親密度が変化する。急変しないよう1セッションあたりの変化量に上限あり。変化の履歴はすべて閲覧可能
- **自律会話**: 2〜6体のペルソナがテーマに沿って会話し、お互いの記憶と関係性に反映される
- **透明性**: 記憶と人格の中身をいつでも閲覧・編集・削除できる
- **エクスポート/インポート**: 育てたペルソナ(初期設定・人格・記憶。会話履歴は選択制)を1つのJSONファイルに書き出し、別のPCの Personacle に取り込める。ペルソナの編集画面から書き出し、サイドバーの「ファイルから取り込む」で取込

### v0.2 で追加 — 「育てている実感」を深める

- **話しかけ**: チャットを開くと、前回からの経過や記憶を踏まえてペルソナの方から話しかけてくる(設定でオフにできる)
- **ムード**: 会話の内容で変わる短期的な気分を持ち、応答のトーンに反映される。時間とともに平常へ戻る。「人格」タブで現在の気分を確認できる
- **記憶の整理**: 似た記憶が増えると自動で要約・統合され、熟成していく(統合の由来はたどれる)
- **日記**: 対話した日について、ペルソナ自身の視点の日記が生成される(「日記」タブ)
- **成長ダッシュボード**: 性格5軸と親密度の推移を折れ線グラフで閲覧できる(「成長」タブ)
- **関係図**: 全ペルソナとユーザーの関係を、親密度の強さつきの図で俯瞰できる(サイドバー「関係図」)
- **記憶の検索**: キーワードと種別で記憶を絞り込める
- **グループチャット**: ユーザーと2〜6体のペルソナで会話。文脈に応じて自動で1体が応答し、`@名前` で指名もできる(サイドバー「グループチャット」)

## 動作環境

- Windows 11 (初版)
- メモリ 16GB 以上を推奨
- GPU は必須ではありません (推奨モデル gemma4 は CPU のみでも動作確認済み。ただし初回のモデル読み込みに数十秒かかります)
- OpenAI 互換 API を持つローカル推論エンジン (Ollama / llama.cpp server / LM Studio)
- 対話は日本語を想定しています (他言語での品質は保証対象外)

## セットアップ手順

### 1. Ollama を導入する

1. [ollama.com](https://ollama.com/) からインストーラを取得してインストール
2. モデルを取得する (ターミナルで実行):

   ```sh
   ollama pull gemma4            # チャット用 (推奨: 8B。日本語の口調維持と速度のバランスが良い)
   ollama pull nomic-embed-text  # 記憶の想起用 (推奨)
   ```

   メモリに余裕がある環境 (24GB以上) では、チャット用に `gpt-oss:20b` も選べます
   (記憶の抽出がより網羅的になる一方、日本語の口調がやや硬く、ロードに時間がかかります)。

### 2. Personacle を起動する

開発版の起動:

```sh
npm install
npm run tauri dev
```

配布用ビルド:

```sh
npm run tauri build
```

### 3. 接続を設定する

1. アプリの「設定」を開く
2. 接続先が `http://127.0.0.1:11434` (Ollama の既定) であることを確認
3. チャットモデルと埋め込みモデルの名前を入力し、「接続確認」で全項目が ✓ になることを確認

あとはペルソナを作って話しかけるだけです。会話を終えると記憶の整理が始まり、「記憶」「人格」タブで中身を確認できます。

## データの保存場所

すべてのデータは以下に保存されます (SQLite 1ファイル):

```text
%APPDATA%\com.personacle.app\personacle.db
```

ログは同フォルダの `logs\` に14日分保持されます。ログに会話本文は記録されません。

## アーキテクチャ概要

Tauri 2 (Rust コア + WebView フロントエンド)。推論は OpenAI 互換 API 経由で外部エンジンに委譲。永続化は SQLite (WAL)。記憶抽出・人格評定はセッション終了後にバックグラウンドで実行され、対話の応答速度に影響しません。強制終了しても次回起動時に未処理の会話を自動回収します。
