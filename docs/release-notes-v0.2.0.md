# Personacle v0.2.0

「育てている実感」を深める大型アップデートです。ペルソナが時間の経過を認識して自分から話しかけ、気分を持ち、記憶を熟成させ、日記を書くようになりました。あわせてグループチャットと成長の可視化を追加しています。すべての処理と保存は引き続きあなたのPCの中で完結します。

## 新機能

### 育成体験 (テーマA)

- **話しかけ (FR-20/21)**: チャット画面を開くと、前回対話からの経過時間や記憶(交わした約束など)を踏まえて、ペルソナの方から会話を切り出します。設定でオフにできます。
- **ムード (FR-24/25)**: 会話の内容で変動する短期的な気分を持ち、応答のトーンに反映されます。対話のない時間の経過で平常へ戻ります(半減期による減衰)。長期的な性格・親密度とは独立で、気分が性格を直接書き換えることはありません。現在の気分は「人格」タブで確認できます。
- **記憶の整理・統合 (FR-22/23)**: 同じ話題の記憶が増えると、自動で要約・一般化した「統合記憶」が作られ、元の記憶はアーカイブされます。統合の由来はいつでもたどれ、誤った統合は記憶の削除時に元へ戻せます。
- **日記 (FR-26/27)**: 対話した日について、ペルソナ自身の視点で書かれた日記が生成されます。「日記」タブで日付順に読めます。

### 可視化 (テーマB)

- **成長ダッシュボード (FR-29)**: 性格5軸と相手ごとの親密度の推移を折れ線グラフで表示します(「成長」タブ)。
- **関係図 (FR-30)**: 全ペルソナとユーザーの関係を、親密度の強さを線の太さで表した図で俯瞰できます(サイドバー「関係図」)。ノードから各ペルソナの詳細へ移動できます。
- **記憶の検索・絞り込み (FR-28)**: 記憶をキーワードと種別(事実・出来事・約束・感想)で絞り込めます。

### 対話形態の拡張 (テーマC)

- **グループチャット (FR-31〜34)**: ユーザーと2〜6体のペルソナで会話できます。あなたの発話に対し、文脈に応じて選ばれた1体が応答します。`@名前` で特定のペルソナを指名でき、ペルソナ同士が続けて話すこともあります。会話は各参加者の記憶・関係性に反映されます。
- **自律会話の停滞対策 (FR-35)**: 自律会話で同じ話題の繰り返しを検出すると、司会役が新しい話題を提案して会話を続けます。

## 動作要件

v0.1.0 と同じです (Windows 11 / メモリ16GB以上推奨 / GPU不要 / Ollama + gemma4 + nomic-embed-text)。埋め込みモデル (nomic-embed-text) は記憶の想起に加え、v0.2 の記憶統合と停滞検出にも使われます。未導入でもこれらの機能は静かにスキップされ、対話自体は動作します。

## データの互換性

- 既存の v0.1.x のデータベースは、起動時に自動でスキーマ v2 へ移行されます (ムード・話しかけ・記憶統合の列と、日記・ムード履歴のテーブルを追加)。**既存のペルソナ・記憶・会話履歴・人格はそのまま保持されます。**
- 移行の直後に、既存の記憶を対象とした初回の整理が一度だけ実行されます。
- エクスポートファイルの形式も更新されました。v0.2 は旧形式のファイルも取り込めます(不足する項目は既定値で補完)。

## 既知の制限

- v0.1.0 と同じく、対話は日本語のみを対象としています。
- 記憶統合・ムード評定・話者選択の品質は使用するモデルに依存します。設定の各しきい値(統合の類似度、ムードの半減期、停滞判定など)は調整可能です。
- 話しかけはアプリ内(チャットを開いたとき)に限られます。アプリを閉じている間のOS通知による通知は行いません。
