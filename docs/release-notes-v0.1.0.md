# Personacle v0.1.0 (初版)

記憶と成長する人格を持つ、ローカルAIキャラクター育成アプリの初リリースです。すべての処理と保存はあなたのPCの中で完結します。

## 機能

- ペルソナの作成 (性格・口調・価値観の設定、性格5軸のAI評定つき)
- 記憶と人格を反映したストリーミングチャット
- 会話からの記憶の自動抽出と、意味検索による想起
- 経験による性格・親密度の成長 (急変しない変化量上限つき) と変化履歴の閲覧
- 2〜6体のペルソナによる自律会話 (会話は各ペルソナの記憶・関係性に反映)
- 記憶・人格の閲覧・編集・削除
- ペルソナのエクスポート/インポート (育てたキャラの共有)

## 動作要件

- Windows 11 / メモリ16GB以上推奨 (GPU不要)
- [Ollama](https://ollama.com/) と推奨モデル:

  ```sh
  ollama pull gemma4
  ollama pull nomic-embed-text
  ```

## インストール

`Personacle_0.1.0_x64-setup.exe` (NSIS) または `Personacle_0.1.0_x64_en-US.msi` を実行してください。

> **注意**: 本リリースはコード署名をしていないため、初回実行時に Windows SmartScreen の警告が表示されます。「詳細情報」→「実行」で続行できます。

## 既知の制限

- 対話は日本語のみを対象としています (他言語は保証対象外)
- CPUのみの環境では初回のモデル読み込みに数十秒かかります
